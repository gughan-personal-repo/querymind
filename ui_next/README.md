# QueryMind UI

A professional, production-grade chat interface for query analysis.

## Setup

1. **Install dependencies**:
   ```bash
   npm install
   ```

2. **Start the Development Server**:
   ```bash
   npm run dev
   ```

3. **Open Browser**:
   Navigate to the URL shown (usually `http://localhost:5173`).

## Architecture

- **Frontend**: React, Vite, Tailwind CSS
- **Backend**: FastAPI (running on port 8000)
- **Design**: Dark mode, Glassmorphism

## Features

- **Hybrid Search**: Intelligently searches tables and columns.
- **SQL Visualization**: Syntax highlighted code blocks.
- **Execution**: Run queries directly against BigQuery.
- **Export**: Download results (coming soon).
